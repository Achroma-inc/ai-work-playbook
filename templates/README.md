# テンプレートの使い方

このディレクトリのファイルを各プロジェクトのリポジトリにコピーして、
ドメイン知識と設計判断を蓄積する。導入手順はリポジトリルートの
[README.md](../README.md) を参照。

## 中身

- `CLAUDE.md` — 各プロジェクトのルートに置く正本。エージェント向けの運用ルール(規範のみにコンパクト化)
- `docs/README.md` — 人間向けの運用ガイド。背景・ファイルの役割・使い分けの解説
- `docs/knowledge.md` — 案件固有のドメイン知識を蓄積する雛形
- `docs/adr/0000-template.md` — 設計意思決定記録(ADR)の雛形

エージェントが従う運用ルールは `CLAUDE.md` に、人間向けの解説は `docs/README.md` に分けてある。
導入後の運用を理解したい人は `docs/README.md` を読む。
